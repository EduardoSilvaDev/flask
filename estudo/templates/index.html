Esta é uma aplicação simples que demonstra o uso de Flask para renderizar templates e exibir informações dinâmicas. Explore as rotas e veja como os dados são passados para os templates.

A aplicação inclui uma rota principal que exibe uma mensagem de boas-vindas e informações do usuário, além de uma rota adicional para exibir uma tabela de estados e capitais.
